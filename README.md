🌱 Agro Sustentável 2026

Projeto desenvolvido para o Concurso Agrinho 2026 – Categoria Programação, utilizando HTML, CSS e JavaScript para promover a conscientização sobre a sustentabilidade no agronegócio de forma educativa, acessível e interativa.

🌾 Tema do Concurso

Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente

📌 Sobre o Projeto

O projeto Agro Sustentável 2026 foi criado com o objetivo de mostrar como o agronegócio e a preservação ambiental podem caminhar juntos.

Por meio de um site moderno e interativo, os visitantes podem aprender sobre a importância do uso consciente dos recursos naturais, conhecer práticas sustentáveis e testar seus conhecimentos através de um quiz educativo.

A proposta busca demonstrar que produzir alimentos de forma responsável é fundamental para garantir qualidade de vida, desenvolvimento econômico e um futuro sustentável para as próximas gerações.

🎯 Objetivo

O principal objetivo do projeto é conscientizar os visitantes sobre a importância da sustentabilidade no campo, destacando a necessidade de preservar o meio ambiente sem deixar de atender às demandas da produção agrícola.

Além disso, o site busca incentivar a reflexão sobre pequenas atitudes que podem contribuir para a conservação da natureza e para o desenvolvimento sustentável.

🧭 Funcionalidades do Site

O projeto conta com diversas funcionalidades para tornar a experiência mais dinâmica e acessível:

🏠 Página inicial com apresentação do tema;
🌱 Seções informativas sobre sustentabilidade e meio ambiente;
🎮 Quiz interativo com perguntas educativas;
📊 Barra de progresso durante o quiz;
🏆 Sistema de pontuação ao final da atividade;
♿ Recursos de acessibilidade;
🔠 Opção para aumentar a fonte;
🔡 Opção para diminuir a fonte;
🌓 Modo de alto contraste;
📱 Layout responsivo para celulares, tablets e computadores.

Pensando em oferecer uma experiência mais inclusiva, o projeto possui recursos que auxiliam diferentes usuários durante a navegação:
